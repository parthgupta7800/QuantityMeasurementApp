## 🚀 Quantity Measurement App - Use Case Tracking

This repository contains my implementation of the **Quantity Measurement App** as part of the BridgeLabz Training Program. Each Use Case (UC) represents a step-by-step enhancement of the application.

* **UC2** → Inch Equality
* **UC3** → Generic Length

* **UC6** → Unit Addition
* **UC7** → Target Unit Addition
* **UC8** → Standalone Unit

* **UC10** → Generic Quantity
* **UC11** → Volume Measurement

* **UC13** → Centralized Arithmetic Logic
